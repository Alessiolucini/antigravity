# Directives (Livello 1)

Questa cartella contiene le **SOP (Standard Operating Procedures)** scritte in Markdown.

## Cosa sono le Directives?

Le directives definiscono:
- **Obiettivi**: Cosa deve essere raggiunto
- **Input**: Quali dati sono necessari
- **Tool/Script**: Quali script di `execution/` utilizzare
- **Output**: Cosa viene prodotto
- **Casi limite**: Come gestire situazioni particolari

## Come creare una Directive

```markdown
# Nome della Directive

## Obiettivo
Descrizione chiara dell'obiettivo.

## Input
- Input 1
- Input 2

## Script da utilizzare
- `execution/script_name.py`

## Output
- Descrizione dell'output atteso

## Casi limite
- Situazione 1: Come gestirla
- Situazione 2: Come gestirla
```

## Regole
- Le directives sono **documenti vivi**: aggiornale quando impari qualcosa di nuovo
- Scrivi istruzioni chiare, come le daresti a un dipendente di medio livello
- Non creare nuove directives senza chiedere prima all'utente
